### Option 1: GitHub Pages (Recommended - Free)

1. **Create GitHub Account**: Go to https://github.com and sign up (free)

2. **Create New Repository**:
   - Click "New repository"
   - Name it: `national-parks-planner`
   - Make it Public
   - Check "Add a README file"
   - Click "Create repository"

3. **Upload Your Files**:
   - Click "uploading an existing file"
   - Drag and drop your `index.html` file
   - Commit changes

4. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Click Save

5. **Get Your Link**:
   - Your site will be live at: `https://yourusername.github.io/national-parks-planner`
   - Share this link with anyone!

### Option 2: Netlify (Free with Drag & Drop)

1. **Go to Netlify**: Visit https://netlify.com
2. **Drag & Drop**: Drag your `index.html` file to the deploy area
3. **Get Link**: Netlify gives you a random URL like `https://amazing-parks-123.netlify.app`
4. **Custom Domain** (optional): Change to a custom name in site settings

### Option 3: Vercel (Free)

1. **Go to Vercel**: Visit https://vercel.com
2. **Sign up** with GitHub/Google
3. **Import Project**: Upload your `index.html`
4. **Deploy**: Get instant URL like `https://national-parks-planner.vercel.app`

### Option 4: Surge.sh (Command Line - Free)

```bash
npm install -g surge
cd your-project-folder
surge
# Follow prompts to get: https://your-chosen-name.surge.sh
```
